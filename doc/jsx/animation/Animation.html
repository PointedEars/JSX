jsx.animation.Animation

  • Object
    • jsx.animation.Animation

An Animation is a set of one or more Timelines being played in parallel when the animation is played. Each Timeline is supposed to animate another target object (but that is not required).

Prototype Properties

target : Object = null
timelines : Array = []

Constructor

jsx.animation.Animation(oParams : Object)

Arguments

oParams : Object
The parameters of this animation. Supported parameters, to be defined as properties of oParams, include:
target : Object
Reference specifiying the object to animate. If specified, the timelines of this animation automatically inherit its target; if not specified for the animation, each timeline of an animation MUST specify its target.
timelines : Array
Timelines of this animation (required)

Throws

jsx.InvalidArgumentError
if no parameters were specified

Methods

play()

Plays the animation by playing all of its Timelines.

Throws

jsx.animation.NoTimelineError

stop()

Stops the animation by stopping all of its Timelines.

Throws
