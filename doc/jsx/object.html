Methods of jsx

dmsg(sMsg[, sType = "log"]) : boolean

Prints debug messages to the script console.

NOTE: This method has previously been provided by {@link debug.js}; optimizations in code reuse moved it here.

Arguments

sMsg : string
Message to be printed
sType : string
Type of the message. Supported values include "log" (default), "info", "warn", and "debug". If a script console does not support a message type, the default value is used.

Returns

true if it was possible to cause the message to be printed; false otherwise.

Methods of jsx.object

addProperties(oSource : Object[, iFlags[, oOwner]])

Adds/replaces properties of an object

Arguments

oSource : Object
Object specifying the properties to be added/replaced. The name of each property serves as the name for the property of the target object, its value as the value of that property.
iFlags : number
Flags for the modification, see {@link Object#ADD_OVERWRITE ADD_*} and {@link Object#COPY_ENUM COPY_*}.
oOwner : Object
If provided, used as target object instead of the calling object. This makes it possible to call the method without an explicit calling object.
